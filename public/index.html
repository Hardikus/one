Hello, ClojureScript!

ClojureScript is brand new and figuring out how to get started without resorting to a framework is a little tough. This project provides a starting point for ClojureScript development with code reloading, templating, a browser-connected REPL, and easy advanced compilation testing.

Learn more »

Design

Let's face it, designers don't like working in Clojure. They need to have a way to work with the tools they are comfortable with and we need a way to pull their work into an application. As a designer, run ./script/run and work on the files in the templates directory. Templates and snippets will be extracted from these files at compile time.

Development

While you are working on a project you will want to use a browser-connected REPL and you will want to avoid using the Google Closure compiler. It is helpful to keep all of your input file separate for debugging. Development mode gives you all of this. This is where you will do most of your work.

Production

Advanced compilation will remove dead code, minify what is left and emit a single JavaScript file. It is good to occationally test that your code will work in advanced mode. This view of your applicaiton is advanced compiled every time you reload the page.
